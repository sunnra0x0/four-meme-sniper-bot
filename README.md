## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- MongoDB
- Redis
- Git
- BNB Chain wallet

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd @four-meme-sniper
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment**
   ```bash
   cp config/env.example config/.env
   # Edit config/.env with your settings
   ```

4. **Start services**
   ```bash
   # Start the bot
   npm run start:bot
   
   # Start web interface
   npm run start:web
   ```
